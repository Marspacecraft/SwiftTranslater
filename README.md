# SwiftTranslater
 swift文件注释翻译

# 安装
-安装[nlohmann-json](https://github.com/nlohmann/json)  
-安装python3
`
make build && cd build  
cmake ..  
make  
make install
`
# 用法  
## 分析文件  
`swifttrans SwfitUI.swift`  
## 查询关键字  
`swifttrans "withTransaction<Result>"`
